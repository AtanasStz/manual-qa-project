## Project Overview

This repository contains a QA testing portfolio project created to demonstrate practical software testing skills using industry-standard tools and workflows.
The project focuses on testing an e-commerce website and includes test planning, test case design, test execution, defect reporting, and test management


### Objectives:

- Analyze functional requirements

- Design comprehensive test cases

- Execute manual tests

- Identify and document defects

- Maintain test traceability

- Simulate a real-world QA workflow


### Test Cases:

#### Structured test cases created using industry-standard QA practices:

* Positive Test Scenarios

* Negative Test Scenarios

* Boundary Value Scenarios

* Edge Cases

### Bug Reports

#### Each bug report includes:

* Summary
 
* Preconditions
 
* Steps to Reproduce
 
* Expected Result
 
* Actual Result
 
* Severity
 
* Priority
